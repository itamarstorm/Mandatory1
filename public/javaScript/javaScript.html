JavaScript

JavaScript er mest anvedt til webudvikling. JavaScript bliver oftes udført clientside af en browser som fx firefox eller chrome. Pga navnet skulle man tro JavaScript var skrevet i java men faktisk er JavaScript en standard der kan implemteres med næsten alle programeringssprog. Chromes JavaScript egnine V8 som nodeJS er bygget op omkring er fx skrevet i C++

Functions

Functions i javaScript er blokke af kode som udfører en specific opgave og bliver udført når den bliver kaldt

Syntax

function name(parameter1, parameter2, parameter3) { // code to be executed }

Arrow Functions

Arrow functions fungere ligesom en normal function men har en kortere syntax

Syntax

const x = (x, y) => x + y

Callback Functions

I javaScript bliver funtioner udført i den rækkefølge de bliver kaldt. Hvis man vil have bedre kontrol over hvornår functions bliver kaldt kan man bruge callback functioner:

function myDisplayer(some) { document.getElementById("demo").innerHTML = some; } function myCalculator(num1, num2, callback) { let sum = num1 + num2; callback(sum); } myCalculator(5, 5, myDisplayer);

En callback function er altså en function der bliver passed som en argument og kaldt af en anden function

Declaring variables

Syntax

const a = 100 let a = 200

For at declare variabler i javaScript kan man bruge entet let eller const. Du kan ikke ændre på en const reference men man kan godt ændre på værdierne i en const.

Ex:

// You can create a const object: const car = {type:"Fiat", model:"500", color:"white"}; // You can change a property: car.color = "red"; // You can add a property: car.owner = "Johnson";

Men ikke:

const car = {type:"Fiat", model:"500", color:"white"}; car = {type:"Volvo", model:"EX60", color:"red"}; // ERROR

Syntax

let array_name = [item1, item2, ...];

Objects

Syntax

let car = {type:"Fiat", model:"500", color:"white"};

Object er variable der indeholder flere værdier. Værdierne er angivet i name:value pairs. Der er to måder at tilgå et objects værdier:

objectName.propertyName objectName["propertyName"]
