jQuery

jQuery et en javascript bibliotek der gør det nemmere at udvikle hjemmesider.

jQuery er meget udbredt og i maj 2019 blev der blandt de 10 millioner mest populære hjemmesider brugt jQuery til at udvikle 73% af dem

jQuery indholder metoder der forsimpler integrationen mellem javascript og HTML. Heriblandt metoder der gør fx. AJAX nemmere at bruge

Howto

For at bruge jQuery skal vi implemtere det i vores HTML document. Vi kan gøre dette via en CDN(Content delivery network). En af de mest udbredte og hurtige jQuery CDN er googles. Ved at gå ind på https://developers.google.com/speed/libraries og kopiere linket under jQuery, kan vi inkludere det i vores HTML og dermed bruge jQuery

Og kan impoteres således:

Ex:

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>

Nu kan jQuery biblioteket bruges.

jQuery Selectors

jQuery selectors giver muligheder for at manipulere HTML elementer, ved hjælp af blandt andet elementets navn eller id

For eksempel kan man få fat i alle <p> således:

$("p")

Dette viser hvordan vi kan få fat i html elementer via elementes tag, men hvis vi vil ha fat i enkelte elementer kan vi gøre det via elementets id.

Det kan gøres via #id selector.

Ex:

$("#test")

Kan også bruges på class for at få fat i alle elementer der tilhøre en class:

Ex:

$(".test");

jQuery events

Events er når en bruger udfører en "action" på en side. jQuery indeholde metoder som kan bruges til at håndtere disse events

Et click på en knap kan fx være et event og kan håndteres på denne måde:

Ex:

$("p").click(function(){ $(this).hide(); });;

I dette tilfælde vil alle p elementer forvinde når man clicker på dem.

jQuery DOM manipulation

jQuery har metoder der gør det mulig at manipulere med DOM(Document Object Model). Her er et par af de mest brugte, samt et eksempel:

Metoder Beskrivelse
append() Indsætter indhold i slutningen af element
before() Indæstter indhold i starten af elements
remove() Fjerner indhold fra DOM

Ex:

$(".inner").append("<p>Test</p>");

Her bruger vi en selector(.inner), så der på alle elementer af klasse inner bliver indsat <p>Test</p>

AJAX

jQuery indeholder også metoder vi kan bruge med AJAX. Fx .load:

Ex:

function changeContentFunction(resource) { $(document).ready(() => { $("#div1").load(resource) }) }

Her henter vi indhold og indsætter den retunerede data ind i #div1 elementet
