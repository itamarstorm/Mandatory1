NodeJS

About

Node.js er et opensource Javascript runtime miljø som bruges til at udvikle wbeapplikationer

Node.js gør det muligt for webudvikler at bruge JavaScript i backenden og skabe dynamiske hjemmesider før de bliver sendt til en brugers browser

Før hen blev JavaScript udført clientside af brugerens browser, men ved hjælp af node.js kan hjemmesider nu udvikles med et enkelt sprog både til backend og frontend.

npm

Node Package Manager er et værktøj som kan bruges til at håndtere de pakker vi bruger til udviklingen af node.js applikationer og generel styring af node projekter

Ved hjælp af npm kan vi nemt installere, samt afinstallere 3 third party pakker til vores node.js projekter. En af de pakker vi har brugt mest undervisningen er

express

som tilbyder metoder der gør det nemmere at opsætte en webservice.

npm console commands

REST API

Det er hurtigt og nemt at opsætte en REST API med node.js, især pga af de mange pakker man kan impotere

Howto

For at bruge node skal programet først installeres. Det kan gøres via dette link. (link)

Når vi opretter et nyt projekt laver vi en ny fil der skal hedde app.js. I denne fil kommer al backend til at være.

Det næste er at lave en package.json fil (Her kan npm init også bruges). Denne json fil indeholder alt metadata omkring vores projekt(name, version, despription osv) samt hvilken dependencies vores projekt bruger.

Det er primært 2 forskellige metoder at tilføje nye pakker til vores projekt. Det kan gøres ved at tilføje pakkens navn samt version til "dependencies" i package.json og herefter bruge npm install.

Ex:

{ "description": "Mandatory assignment for elective node.js", "dependencies": { "cors": "^2.8.5", "express": "^4.17.1" } }

Det kan dog også gøres nemmere ved at bruge npm install <package> som automatisk tilføjer pakken til package.json
