Express

Link

Express er et webframework der hjælper med at organisere MVC-arkitektur på serverside. Det er et af de mest poplære frameworks til node.js. Express giver os mulighed for at håndtere forskellige HTTP requests relativt nemt. Express indeholder funktionaliteter der kan håndtere HTTP requests som fx GET, POST, PUT, PATCH og DELETE

Fx kan man med express hurtigt lave en applikation der kan håndtere et get request på en specific url:

Det første vi skal gøre er at importere express. Denne syntax gælder hver gang man skal importere en pakke til node:

const express = require("express")

Herefter kan vi oprette en variable som vi kalder app, som indeholder alle funktioner fra express:

const app = express()

Laver et endpoint på "/" som håndtere et get request fra en client og starter serveren på port 8081:

app.get("/" , (req, res) => { return res.send("Hello World!") }) app.listen(8081, (error) => { if (error) { console.log("Error starting the server") } console.log("Server started on port", 8081) })

get metoden tager imod en callback function som har 2 parametre. Pga af konventioner kalder man dem altid for req og res

res repræsentere det response som applikation sender når den får et HTTP request og req indeholder de informationer der bliver sendt som query string, parametre, body osv.

Med express kan man nemt tage brug af middleware ved hjælp af .use Fx kan vi bruge express.static som er indbygget i express. Vi kan ved hjælp af dette gøre en mappe public så de er frit tilgængelige. Dette er vigitgt hvis man tager brug af external css documenter, da man ellers ville få en 404 fejl
